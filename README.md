Анализ данных о сне и здоровье.

Цель проекта: Изучение влияния различных факторов на сон.

В этом проекте были проведены:

1) Загрузка данных и их первичный осмотр.
2) Обработка данных.
3) Визуализация данных, с помощью графиков.
4) Различные группировки и агрегации.

Что было использовано:

1) Python (Google Colab).
2) Yandex DataLens.
3) Файл Sleep_health_and_lifestyle_dataset (1) с данными.

Интересные выводы:

1) Чем выше стресс - тем хуже сон.
2) У разных профессий разный сон.
3) У полных людей выше пульс.
